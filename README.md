# Приложение для просмотра погоды.
Реализованы функции:
  1. Просмотр погоды на данный момент.
  2. Просмотр погоды на будущее.
  3. Определение региона и времени пользователя. 
  4. SVG Animation

## Стэк:
  * React
  * Typescript
  * Moment.js
  * MaterialUI

## Запуск:
  * `git clone https://github.com/Nikikwiki/weather-app`
  * `cd weather-app`
  * `npm install`
  * `npm start`
